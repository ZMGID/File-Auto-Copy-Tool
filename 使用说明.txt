# 文件自动复制工具 - 完整版

一个智能的文件复制工具，可以根据文件名中的数字标记自动复制指定数量的文件。

## 📁 文件结构
```
文件自动复制工具/
├── Mac/                          # Mac版本
│   ├── 文件自动复制工具           # Mac可执行文件
│   └── 使用说明.txt              # Mac使用说明
├── Windows/                      # Windows版本
│   ├── 文件自动复制工具_Windows版.bat  # Windows批处理版本
│   ├── 自动打包EXE.bat           # 自动打包脚本
│   └── 使用说明.txt              # Windows使用说明
├── 源码/                         # 源代码和开发文档
│   ├── auto_copy_stl.py          # Python源码
│   ├── Windows_EXE打包指南.md     # EXE打包指南
│   ├── Windows打包说明.md         # 开发说明
│   └── README.md                 # 源码说明
├── STL/                          # 示例文件夹
└── 使用说明.txt                  # 本文件
```

## 🚀 快速开始

### Mac用户
1. 进入 `Mac` 文件夹
2. 双击 `文件自动复制工具`
3. 按提示选择文件夹

### Windows用户
1. 进入 `Windows` 文件夹  
2. 双击 `文件自动复制工具_Windows版.bat`
3. 按提示选择文件夹

## ✨ 主要功能
- 🎯 **智能识别**: 自动识别文件名中的x2、x4等复制标记
- 📁 **全格式支持**: 支持所有文件类型（STL、PDF、JPG、TXT等）
- 🔄 **防重复**: 自动跳过已存在的复制文件
- 🖱️ **拖拽支持**: 支持拖拽文件夹操作
- 🌍 **中文支持**: 完美支持中文路径和文件名

## 📋 使用示例
- `零件_x2_.stl` → 生成1个副本（总共2份）
- `文档_x3_.pdf` → 生成2个副本（总共3份）  
- `图片_x5_.jpg` → 生成4个副本（总共5份）

## 💡 系统要求
- **Mac**: macOS 10.12+ (无需额外安装)
- **Windows**: Windows 7/8/10/11 (无需额外安装)
